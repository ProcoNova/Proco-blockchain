# ProCo Blockchain

ProCo is an experimental educational blockchain project.

⚠️ This repository is for research, learning, and community discussion only.  
⚠️ No production network, live tokens, or financial use yet.

## Current Status
- Core blockchain concepts implemented
- Private multi-node testing completed on AWS
- Code refactoring and simplification in progress
- Documentation and community onboarding started

## Vision
To build a simple, transparent, and understandable blockchain
for learning, experimentation, and future innovation.

More updates coming soon.
